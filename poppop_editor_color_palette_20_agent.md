## 💡 20人が策定した PopPop 公式カラーパレット（Color Standard）

VS Codeや各種エディタにおいて、PopPopコードを開いた瞬間に **「何が関数で、何が変数で、何がパイプラインか」** が0.1秒で脳に伝わる公式配色です。

---

### 🎨 カラーパレット概要（Dark Mode / Light Mode 対応）

| シンボル種別 | 役割・意味 | ダークモード色 (Dark+) | ライトモード色 (Light+) | セマンティック意味 |
| :--- | :--- | :--- | :--- | :--- |
| **関数 / 動詞 (Verbs)** | `Display`, `Map`, `Sort`, `Sum` | 🟡 **サンシャインゴールド (`#FFD700`)** | 🟠 **アンバーオレンジ (`#D19A66`)** | データを加工・出力する **能動的アクション** |
| **変数 / 名詞 (Nouns)** | `scores`, `track`, `item` | 🔵 **アイスシアン (`#4FC1FF`)** | 🔵 **コバルトブルー (`#0066CC`)** | 保持されている **静的データ・値** |
| **パイプライン演算子** | `>>`, `>!>`, `>~>` | 🟣 **ネオンマゼンタ (`#C678DD`)** | 🟣 **ディープパープル (`#7C3AED`)** | データを右へ流す **データハイウェイ** |
| **プロパティ `::`** | `track::title`, `item::price` | 🟢 **ミントグリーン (`#98C379`)** | 🟢 **フォレストグリーン (`#2E7D32`)** | 辞書内部への **アクセスキー** |
| **制御ブロック** | `Catch(err):`, `Filter:`, `..` | 🟧 **コーラルオレンジ (`#E06C75`)** | 🔴 **クリムゾン (`#DC2626`)** | コードの **構造・分岐の囲み** |
| **文字列 & 挿入** | `"text {var}"` | 🟤 **ウォームアンバー (`#CE9178`)** | 🟢 **オリーブグリーン (`#15803D`)** | テキストデータ（中身の `{var}` は青色）|

## 📜 VS Code TextMate スコープ定義 (`poppop.tmLanguage.json`)

エディタのハイライトエンジン（TextMate Grammar）へ登録する標準スコープ名です：

```json
{
  "functions": {
    "name": "entity.name.function.poppop",
    "match": "\\b[A-Z][a-zA-Z0-9_]*\\b",
    "comment": "大文字始まりの関数（動詞）はゴールドカラー"
  },
  "variables": {
    "name": "variable.other.readwrite.poppop",
    "match": "\\b[a-z_][a-zA-Z0-9_]*\\b",
    "comment": "小文字始まりの変数（名詞）はシアンカラー"
  },
  "pipeline_operators": {
    "name": "keyword.operator.pipeline.poppop",
    "match": ">>|>!>|>~>|>\\+>|>\\?>",
    "comment": "パイプライン演算子はマゼンタパープル"
  },
  "double_colon": {
    "name": "keyword.operator.doublecolon.poppop",
    "match": "::",
    "comment": "プロパティ参照 :: はミントグリーン"
  }
}
```

## 🖥️ 実際のVS Code上での見え方イメージ

エディタ上でコードを開くと、カッコ `()` がなくても以下のように完璧に一目で識別できます：

```poppop
// 🔵変数    🟣パイプ 🟡関数 (変数)
   raw_tracks  >>     Map    (track):
       (track, "rating", (Track::rating >> Num >> Catch(err): 0.0 ..)) >> Set.
   .. >> Filter(track):
       track::genre == "Pop" && track::plays >= 1000000.
   .. >> Sort(track):
        -track::plays.
   .. >> List >> new_tracks.

//  🟡黄色＝関数 (Map, Filter, Sort, Display)
//  🔵青色＝変数 (raw_tracks, track, new_tracks)
//  🟣紫色＝パイプライン (>>)
//  🟢緑色＝プロパティ (::genre, ::plays)
```

## 🏆 20人委員会の結論

**「カッコ `()` がないPopPopにとって、配色（Syntax Highlighting）は単なる飾りではなく、視認性を完成させる最重要インターフェースである。」**

サンシャインゴールド（関数）とアイスシアン（変数）のカラーコントラストにより、開発者はカッコを一切打つことなく、0.1秒でコードの構造を把握できるよう策定いたしました！🍿🎈
