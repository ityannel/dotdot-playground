# パート3: 関数、モジュール、および組み込み関数

## 関数定義 (`new`)
Dot Dotでは、新しい関数は`new`キーワードを使用し、その後に関数名、パラメーター、およびコードブロックを続けて定義します。

```
new greet(name) {
    "Hello, " + name >> Display
}
```

## モジュール (`Import`)
組み込み関数の`Import`を使用すると、他の`.pop`ファイルを現在の環境にインポートして実行できます。

```
"utils.pop" >> Import
```

## コア組み込み関数

### 入出力と制御

| 関数 | 入力 | 出力 | 動作 | コード例 |
|----------|-------|--------|----------|--------------|
| `Display` | 任意の型 | 入力と同じ | 入力を標準出力に表示する | `"Hello" >> Display` |
| `Input` | `str` (プロンプト) | `str` | ユーザーに入力を促す | `"Name? " >> Input` |
| `Return` | 任意の型 | なし (中断) | 関数から値を返す | `42 >> Return` |
| `Break` | 任意の型 | なし (中断) | ループから抜け出す | `1 >> Break` |
| `Sleep` | `int` または `float` | 入力と同じ | 指定された秒数だけ実行を一時停止する | `2 >> Sleep` |
| `Fetch` | `str` (URL/パス) | `str` | ファイルまたはURLのコンテンツを取得する | `"http://x.com" >> Fetch` |
| `PostFetch` | `[url, headers, body]` | `str` | POSTリクエストを実行する | `["url", {}, ""] >> PostFetch` |
| `Serve` | `[port, handler]` | `port` | `handler`を使用して`port`でHTTPサーバーを起動する | `[8080, "myHandler"] >> Serve` |
| `WriteFile` | `[content, path]` | `content` | コンテンツをファイルに書き込む | `["data", "out.txt"] >> WriteFile` |
| `Debug` | 任意の型 | 入力と同じ | `[DEBUG]`と共に、入力のJSON/文字列表現を出力する | `[1, 2] >> Debug` |
| `Throw` | `str` | なし (中断) | ユーザーエラーをスローする | `"Error!" >> Throw` |

### キャストと型変換

| 関数 | 入力 | 出力 | 動作 | コード例 |
|----------|-------|--------|----------|--------------|
| `Int` | `str`, `float`, `bool`| `int` | 入力を整数に変換する | `"42" >> Int` |
| `Str` | 任意の型 | `str` | 入力を文字列に変換する | `42 >> Str` |
| `Bool` | 任意の型 | `bool` | 入力をブール値に変換する | `1 >> Bool` |
| `Type` | 任意の型 | `str` | 入力の型名を返す | `42 >> Type` |
| `Array` | `int` または `[int,...]` | `list` | 指定された次元のゼロ埋め配列を作成する| `[2, 2] >> Array` |
| `ToJson` | `dict` または `list` | `str` | データ構造をJSON文字列に変換する | `{"a": 1} >> ToJson` |
| `FromJson`| `str` | `dict`/`list` | JSON文字列を解析してデータ構造にする | `"{\"a\": 1}" >> FromJson`|

### 数学と数値

| 関数 | 入力 | 出力 | 動作 | コード例 |
|----------|-------|--------|----------|--------------|
| `Add` | `[num, num]` | `num` | 2つの数値を加算する | `[1, 2] >> Add` |
| `Sum` | `[num, ...]` | `num` | 数値のリストを合計する | `[1, 2, 3] >> Sum` |
| `Max` | `[num, ...]` | `num` | リスト内の最大値を返す | `[1, 2, 3] >> Max` |
| `Min` | `[num, ...]` | `num` | リスト内の最小値を返す | `[1, 2, 3] >> Min` |
| `Round` | `num` | `int` | 数値を最も近い整数に四捨五入する | `3.14 >> Round` |
| `Abs` | `num` | `num` | 数値の絶対値を返す | `-5 >> Abs` |
| `Random` | `[min, max]` | `int` | minからmaxまでのランダムな整数を生成する | `[1, 10] >> Random` |

### リストと文字列の操作

| 関数 | 入力 | 出力 | 動作 | コード例 |
|----------|-------|--------|----------|--------------|
| `Length` | `list` または `str` | `int` | リストまたは文字列の長さを返す | `[1, 2] >> Length` |
| `Sort` | `list` | `list` | リストを昇順にソートする | `[3, 1] >> Sort` |
| `Reverse` | `list` または `str` | `list`/`str` | リストまたは文字列を反転させる | `"abc" >> Reverse` |
| `Uppercase`| `str` | `str` | 文字列を大文字に変換する | `"a" >> Uppercase` |
| `Lowercase`| `str` | `str` | 文字列を小文字に変換する | `"A" >> Lowercase` |
| `Split` | `[str, delim]` | `list` | 文字列を区切り文字で分割する | `["a,b", ","] >> Split` |
| `Replace` | `[str, old, new]`| `str` | `old`の出現箇所を`new`に置換する | `["a b", " ", "_"] >> Replace` |
| `Join` | `[list, delim]` | `str` | リストを区切り文字で結合する | `[[1, 2], "-"] >> Join` |
| `Get` | `[dict/list, key]` | 任意 | キーまたはインデックスで値を取得する | `[[1, 2], 0] >> Get` |
| `Set` | `[dict/list, k, v]`| `dict`/`list`| キーまたはインデックスに値を設定する | `[[1, 2], 0, 5] >> Set` |
| `Contains` | `[dict/list, v]` | `bool` | ターゲットに値が含まれているかを確認する | `[[1, 2], 1] >> Contains` |
| `Append` | `[list, item]` | `list` | アイテムをリストの末尾に追加する (新しいリストを返す) | `[[1], 2] >> Append` |
| `Slice` | `[list, st, end]`| `list` | 開始インデックスから終了インデックスまでリストをスライスする | `[[1,2,3], 0, 2] >> Slice` |
| `Format` | `[template, ...]`| `str` | 引数を使用して文字列をフォーマットする | `["{} {}", 1, 2] >> Format` |
| `Group` | `list` または `[list, prop]`| `dict` | プロパティ/値でリストの要素をグループ化する | `[arr, "id"] >> Group` |
| `Lazy` | `list` | `iterator` | リストを遅延イテレータに変換する | `[1, 2] >> Lazy` |
| `Take` | `int` または `[iter, n]`| `list` | イテレータ/現在のストリームから`n`個の要素を取得する | `[iter, 2] >> Take` |
| `Count` | `int` | `iterator` | 数値から無限イテレータを生成する | `1 >> Count` |
