# Part 4: 高度な組み込み関数 (Advanced Built-ins)

## コレクションとイテラブルの組み込み関数 (Collections & Iterables Built-ins)

| 組み込み関数 | 入力 | 出力 | 動作 | コード例 |
| :--- | :--- | :--- | :--- | :--- |
| **Array** | 数値 または 数値のリスト | 配列 (ネストされたリスト) | 0で初期化されたN次元の配列を作成します。単一の数値 `N` はサイズNの配列を作成します。リスト `[N, M]` は `N x M` の配列を作成します。 | `[2, 3] >> Array` (出力: `[[0,0,0], [0,0,0]]`) |
| **Range** | 数値 または `[start, end]` | 数値のリスト | 整数のリストを生成します。単一の数値 `N` が指定された場合、`1` から `N` まで (終端を含む) を生成します。`[start, end]` の場合、`start` から `end` まで (終端を含む) を生成します。 | `[1, 5] >> Range` (出力: `[1, 2, 3, 4, 5]`) |
| **Slice** | `[target_list, start, end]` | リスト | `start` インデックスから始まり、`end` インデックスの前で終わるリストの一部を抽出します。 | `[[10, 20, 30], 0, 2] >> Slice` (出力: `[10, 20]`) |
| **Append** | `[target_list, item]` | リスト | `target_list` の末尾に `item` を追加した新しいリストを返します。 | `[[1, 2], 3] >> Append` (出力: `[1, 2, 3]`) |
| **Group** | リスト または `[target_list, prop_name]` | 辞書 | リストの要素をグループ化します。リストのみが指定された場合、要素自体でグループ化します。`[list, prop_name]` が指定された場合、指定されたプロパティで辞書をグループ化します。 | `[[{id: 1}, {id: 1}], "id"] >> Group` (出力: `{"1": [{id: 1}, {id: 1}]}`) |
| **Contains** | `[target_list, item]` | ブール値 | `item` が `target_list` 内に存在する場合、`true` を返します。 | `[[1, 2, 3], 2] >> Contains` (出力: `true`) |
| **Get** | `[target_collection, key]` | 任意 | 指定された `key` (辞書の場合) または インデックス (リストの場合) の値を取得します。 | `[{"a": 1}, "a"] >> Get` (出力: `1`) |
| **Set** | `[target_collection, key, value]` | コレクション | `key` または インデックスに `value` を設定してコレクションを変更し、変更されたコレクションを返します。 | `[[10, 20], 0, 99] >> Set` (出力: `[99, 20]`) |
| **Length** | リスト、文字列、または 辞書 | 数値 | コレクション内の要素数、または文字列内の文字数を返します。 | `[1, 2, 3] >> Length` (出力: `3`) |
| **Sort** | リスト | リスト | 要素を昇順にソートした新しいリストを返します。 | `[3, 1, 2] >> Sort` (出力: `[1, 2, 3]`) |
| **Reverse** | リスト または 文字列 | リスト または 文字列 | リストの要素または文字列の文字の順序を反転させます。 | `[1, 2, 3] >> Reverse` (出力: `[3, 2, 1]`) |
| **Min** | 数値のリスト | 数値 | リスト内の最小の数値を返します。 | `[5, 1, 9] >> Min` (出力: `1`) |
| **Max** | 数値のリスト | 数値 | リスト内の最大の数値を返します。 | `[5, 1, 9] >> Max` (出力: `9`) |
| **Sum** | 数値のリスト | 数値 | リスト内のすべての数値の合計を計算します。 | `[1, 2, 3] >> Sum` (出力: `6`) |

## 文字列の組み込み関数 (String Built-ins)

| 組み込み関数 | 入力 | 出力 | 動作 | コード例 |
| :--- | :--- | :--- | :--- | :--- |
| **Replace** | `[target_str, old, new]` | 文字列 | `target_str` 内のすべての `old` 部分文字列を `new` 部分文字列に置換します。 | `["hello", "e", "a"] >> Replace` (出力: `"hallo"`) |
| **Split** | `[target_str, delimiter]` | 文字列のリスト | 指定された `delimiter` を使用して `target_str` を文字列のリストに分割します。 | `["a,b", ","] >> Split` (出力: `["a", "b"]`) |
| **Join** | `[target_list, delimiter]` | 文字列 | `delimiter` で区切って `target_list` の要素を単一の文字列に連結します。 | `[[1, 2], "-"] >> Join` (出力: `"1-2"`) |

## ネットワーク、ファイルシステム、および Web/Serve 組み込み関数 (Network, FileSystem, and Web/Serve Built-ins)

| 組み込み関数 | 入力 | 出力 | 動作 | コード例 |
|---|---|---|---|---|
| **ToJson** | 任意のデータ構造 | JSON 文字列 | 値またはデータ構造を JSON 形式の文字列に変換します。 | `{ "name": "PopPop" } >> ToJson` |
| **FromJson** | JSON 文字列 | データ構造 (辞書、リストなど) | JSON 文字列を解析して対応するデータ構造にします。 | `'{"name": "PopPop"}' >> FromJson` |
| **Fetch** | URL 文字列 または ファイルパス 文字列 | 文字列 (コンテンツ) | GET リクエストを介して指定された HTTP(S) URL からデータをフェッチするか、ファイルパスが指定されている場合はローカルファイルを読み取ります。 | `'https://api.example.com/data' >> Fetch` |
| **PostFetch** | 3つの要素の配列: `[URL, Headers Dict, Body String]` | 文字列 (レスポンスコンテンツ) | ヘッダーとボディのペイロードを使用して、指定された URL に HTTP POST リクエストを送信します。 | `["http://example.com/api", {"Content-Type": "application/json"}, '{"key": "value"}'] >> PostFetch` |
| **WriteFile** | 2つの要素の配列: `[Content String, Path String]` | 文字列 (元のコンテンツ) | UTF-8 エンコーディングを使用して、指定されたパスのファイルに、指定されたコンテンツ文字列を書き込みます。 | `["Hello World", "./test.txt"] >> WriteFile` |
| **Serve** | 2つの要素の配列: `[Port Number, Handler Function Name]` | ポート番号 | 指定されたポートでローカル HTTP サーバーを起動します。受信リクエストは、`method`、`path`、`query`、`body`、および `client` を持つ辞書として、指定されたハンドラー関数に渡されます。ハンドラーは、文字列、または `status`、`headers`、および `body` を持つ辞書を返すことができます。 | `[8080, "MyHandler"] >> Serve` |
| **Import** | ファイルパス 文字列 | ファイルパス 文字列 | グローバル環境内で、指定された Dot Dot スクリプトファイルの内容を読み取り、解析し、評価します。 | `"./my_module.pop" >> Import` |

## 一般的な組み込み関数 (General Built-in Functions)

| 関数 | 入力 | 出力 | 動作 | コード例 |
| :--- | :--- | :--- | :--- | :--- |
| **Type** | 任意の値 (`bool`, `int`, `str`, `list`, `dict`, など) | `str` (例: `'bool'`, `'int'`) | 指定された値のデータ型の文字列表現を返します。 | `Type(True)` >> `'bool'` |
| **Sleep** | 秒数を表す数値 (`int`, `float`) | 同じ入力値 | 指定された秒数だけ実行を一時停止します。 | `Sleep(2)` |
| **Add** | ちょうど2つの要素を持つリスト (例: `[a, b]`) | 2つの要素の合計 | 2つの要素を足し合わせます。 | `Add([5, 3])` >> `8` |
| **Round** | 数値 (`int`, `float`) | `int` | 指定された数値を最も近い整数に丸めます。 | `Round(3.6)` >> `4` |
| **Abs** | 数値 (`int`, `float`) | 数値 (絶対値) | 指定された数値の絶対値を返します。 | `Abs(-5)` >> `5` |
| **Format** | テンプレート文字列で始まり、引数 (または辞書) が続くリスト | フォーマットされた `str` | 指定された引数を使用してテンプレート文字列をフォーマットします。 | `Format(["Hi {}", "Bob"])` >> `'Hi Bob'` |
| **Now** | 任意 (入力を無視します) | `dict` (キー: year, month, day, hour, minute, second) | 現在の日付と時刻を辞書として返します。 | `Now()` >> `{'year': 2026, ...}` |
| **Random** | ちょうど2つの数値のリスト `[min, max]` | `int` | min から max までの (終端を含む) ランダムな整数を生成します。 | `Random([1, 10])` >> `7` |
| **Display** | 任意の値 | 同じ入力値 | 値を標準出力に出力し、変更せずにそのまま返します。 | `Display("Hello")` |
| **Input** | `str` (プロンプトメッセージ) | `str` (ユーザー入力) | プロンプトメッセージを表示して、ターミナル入力のプロンプトをユーザーに出します。 | `Input("Name: ")` |
| **Int** | int にキャスト可能な値 (`str`, `float`) | `int` | 指定された値を整数にキャストします。 | `Int("42")` >> `42` |
| **Str** | 任意の値 | `str` | 指定された値を文字列にキャストします。 | `Str(42)` >> `'42'` |
| **Bool** | 任意の値 | `bool` | 指定された値をブール値にキャストします。 | `Bool(1)` >> `True` |
| **Lazy** | `list` または 任意の値 | イテレータ (`list` の場合)、それ以外は値 | リストを遅延イテレータに変換します。 | `Lazy([1, 2, 3])` |
| **Take** | 数値 `n` または リスト `[source, n]` | 最大 `n` 個の項目の `list` | イテレータまたはリストから最大 `n` 個の項目を消費します。 | `Take([iterator, 2])` |
| **Count** | 開始値としての数値 (`int`) (デフォルトは 1) | 無限イテレータ | 入力から始まる整数の無限シーケンスを生成します。 | `Count(1)` |

## 制御の組み込み関数 (Control Built-in Functions)

| 関数 | 入力 | 出力 | 動作 | コード例 |
| :--- | :--- | :--- | :--- | :--- |
| **Return** | 任意の値 | なし (`ReturnException` をスローします) | 現在の関数の実行を中断し、呼び出し元に値を返します。 | `Return(42)` |
| **Break** | 任意の値 | なし (`BreakException` をスローします) | 現在のループの反復から抜け出し、値を上に渡します。 | `Break("done")` |
| **Throw** | `str` (エラーメッセージ) | なし (`PopPopError` をスローします) | 提供されたメッセージを使用して、ユーザー定義のランタイムエラー (`UserError`) をスローします。 | `Throw("Invalid data")` |
